reading from keyboard
class Scanner, this class is provided from version 5 onwards
package util

System.in is an object attached to keyboard

methods available:
nextInt(): method to read an interger form the keyboard
nextFloat()
nextDouble()
next(): read only the next Word
nextLine(): read the whole next line
nextByte()
nextShort()
nextLong()
nextBoolean()

hasNextInt(): method to confirm if the next value that is coming in is an integer or not, returns boolean value
hasNextFloat()
